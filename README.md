# Prizm Programming Language

Prizm is a beginner-friendly and cool programming language designed to make coding simple and approachable. The language emphasizes clarity and creativity, making it perfect for those new to programming.

## File Extension
- **.pzm**: All Prizm files use the `.pzm` extension.

## Main File
- The main entry point for a Prizm project is `center.pzm`.

## About Prizm

Prizm is a unique programming language where users do not have access to the source files. Instead, they work with compiled and executable files, including all necessary DLLs and the main `.exe` file that executes the program.

### Implementation
- Prizm is written in low-level languages that operate below C, ensuring high performance and efficiency.
